# 💇 Telegram-бот для салона красоты

Полнофункциональный бот с записью клиентов, рассылками и уведомлениями.

---

## 🚀 Быстрый старт

### 1. Создать бота в Telegram
1. Напишите [@BotFather](https://t.me/BotFather) в Telegram
2. Отправьте `/newbot`
3. Придумайте имя и username для бота
4. Скопируйте токен (выглядит как `1234567890:AAF...`)

### 2. Узнать свой Telegram ID
1. Напишите [@userinfobot](https://t.me/userinfobot)
2. Скопируйте ваш ID (числовой, например `123456789`)

### 3. Настроить config.py
Откройте файл `config.py` и заполните:
```python
BOT_TOKEN = "ВАШ_ТОКЕН_ОТ_BOTFATHER"
ADMIN_IDS = [ВАШ_TELEGRAM_ID]  # например [123456789]
SALON_NAME = "Название вашего салона"
SALON_ADDRESS = "ул. Ваша, д. 1"
SALON_PHONE = "+7 (999) 000-00-00"
```

### 4. Установить зависимости
```bash
pip install -r requirements.txt
```

### 5. Запустить бота
```bash
python bot.py
```

## 🔧 Команды администратора

| Команда | Описание |
|---------|----------|
| `/admin` | Главная панель + статистика |
| `/appointments` | Все записи (последние 20) |
| `/today` | Записи на сегодня |
| `/clients` | База клиентов |
| `/broadcast` | Запустить рассылку |
| `/cancel_app 5` | Отменить запись с ID=5 |

## ⚙️ Автоматические функции

- **Напоминания** — каждый день в 18:00 бот отправляет клиентам напоминание о записи на завтра
- **Уведомления** — при каждой новой записи вы мгновенно получаете уведомление с данными клиента

## 💅 Редактирование услуг

Услуги хранятся в базе данных `salon.db`. Чтобы изменить список услуг — отредактируйте блок в `database/db.py`:

```python
await db.execute("""
    INSERT OR IGNORE INTO services (id, name, duration_minutes, price)
    VALUES
        (1, '✂️ Стрижка', 60, 800),
        (2, '🎨 Окрашивание', 120, 2500),
        ...
""")
```

Или используйте любой SQLite-редактор (например [DB Browser for SQLite](https://sqlitebrowser.org/)).

## 🖥️ Запуск на сервере (необязательно)

Для работы 24/7 запустите на VPS через `systemd` или `screen`:

```bash
# Через screen
screen -S salon_bot
python bot.py
# Ctrl+A, затем D — свернуть

# Вернуться к боту
screen -r salon_bot
```

## ❓ Частые вопросы

**Бот не отвечает** — проверьте правильность токена в `config.py`

**Не приходят уведомления** — проверьте правильность вашего ADMIN_ID

**Ошибка при запуске** — убедитесь что установлены зависимости: `pip install -r requirements.txt`
