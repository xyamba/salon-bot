# 💇 Telegram-бот для салона красоты

Профессиональный бот для записи клиентов в салон красоты с удобной админ-панелью.

---

## ⚙️ Настройка

### config.py
```python
BOT_TOKEN = "токен от @BotFather"
ADMIN_IDS = [ваш_telegram_id]  # узнать у @userinfobot

SALON_NAME = "Название салона"
SALON_ADDRESS = "ул. Примерная, д. 1"
SALON_PHONE = "+7 (999) 000-00-00"
WORK_START_HOUR = 10   # начало рабочего дня
WORK_END_HOUR = 21     # конец рабочего дня
```

### Услуги и цены
Редактируются двумя способами:
1. **Через Railway** → PostgreSQL → Data → таблица `services` (как Excel, без кода)
2. **Через код** — в `database/db.py` в блоке `INSERT INTO services`

Формат: `('Название', длительность_в_минутах, цена_в_рублях)`


---

## 🌐 Деплой на Railway

1. Залить код на GitHub
2. Подключить репозиторий в Railway
3. Добавить переменные окружения в **Variables**:
   - `BOT_TOKEN` — токен бота
   - `DATABASE_URL` — строка подключения к PostgreSQL
4. Создать **PostgreSQL** базу: New → Database → PostgreSQL
5. Railway автоматически деплоит при каждом `git push`

---

## 📱 Возможности для клиентов

- Регистрация (имя, фамилия, телефон)
- Запись по категориям услуг
- Слоты с учётом длительности каждой услуги
- Только актуальное время (прошедшие слоты скрыты)
- Просмотр своих записей
- Отмена записи с уведомлением администратора
- Напоминание за день до записи

---

## 🔧 Команды администратора

| Команда | Описание |
|--------|----------|
| `/admin` | Панель управления со статистикой |
| `/today` | Расписание на сегодня |
| `/appointments` | Последние 20 записей |
| `/clients` | База клиентов |
| `/search [имя/телефон]` | Поиск клиента |
| `/stats` | Статистика по услугам и выручка |
| `/confirm [ID]` | Подтвердить запись |
| `/cancel_app [ID]` | Отменить запись с причиной |
| `/broadcast` | Рассылка всем клиентам (текст или фото) |
| `/cleanup` | Удалить прошедшие записи вручную |

---

## 🗄️ База данных

**PostgreSQL** на Railway — данные не слетают при перезапусках и деплоях.

Таблицы:
- `clients` — клиенты
- `services` — услуги и цены
- `appointments` — записи
- `broadcasts` — история рассылок
